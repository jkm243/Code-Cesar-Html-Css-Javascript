Система шифрования Цезаря с ключевым словомя

Особенность системы шифрования Цезаря с ключевым словом - использование ключевого слова для смещения и изменения порядка символов в алфавите подстановки. Для этой системы ключ должен быть составным и содержать некоторое число (например, k) и ключевое слово. Для числа k должно соблюдаться требование 0 < k < m - 1, где m - количество букв в алфавите. Рассмотрим пример шифрования сообщения «ПРИЛЕТАЮ ЗАВТРА» с использованием системы шифрования Цезаря с ключевым словом. Примем k = 5, в качестве ключевого слова будем использовать слово «РАБОТА». Первым этапом шифрования является запись ключевого слова в таблицу подстановок, начиная с буквы исходного алфавита с номером k. Если ключевое слово имеет повторяющиеся буквы, в таблицу подстановок повторно они не записываются
